# 克苏鲁的呼唤 调查员天命五暴躁生成器

本项目基于 Python 和 Tkinter 的图形化（GUI）角色生成工具，设计了一个专用于《克苏鲁的呼唤》（Call of Cthulhu）TRPG的辅助工具，用于快速、有趣地生成调查员的“天命五”（即八大基本属性）。

---

## 项目背景

- 本项目是一个风格独特的角色属性生成器，旨在为 CoC 玩家提供三种不同的属性生成方式，同时融入幽默、互动的界面提示。程序通过 Tkinter 实现图形界面，支持用户选择不同的生成模式，并最终保存心仪的角色属性表。

-主要特点：

 - 完全使用 Python 标准库（tkinter、random、 sys）

 - 三种生成模式：极速生成、分步生成、刺激生成

 - 共生成可生成五套属性供选择（实现天命五功能）

 - 交互式提示与“开发者吐槽”风格提示框

 - 无第三方依赖，开箱即用

## 功能概览

### 极速生成模式
- 一键生成所有未分配的属性
- 适合快速生成角色

### 技能检定系统
- 用户可从列表中选择一个技能进行投掷
- 已分配属性会变灰，防止重复投掷

### 理智（SAN）系统
- 随机选择一个未分配属性
- 分三次点击投掷，每次投掷一个 D6×5
- 投掷完成后给出幽默评价
- 可在界面内投掷完后选择新的技能

### 最终选择界面（Finale）
- 展示五轮生成的全部结果
- 用户可选择一个最终版本
- 确认后展示最终角色卡八维

```text
- main.py

- choice3.py

- resource.py
```

### 运行主程序
```bash
- main.py
```

## 使用说明

### 启动界面
- 输入调查员姓名

### 选择生成模式

- 三种模式说明
 - 极速生成：	一键完成所有属性
 - 分步生成：	自选属性逐个生成
 - 刺激生成：	随机属性分步投掷	

### 生成流程
- 每轮生成一套完整的八大属性
- 共进行五轮生成
- 在最终界面中选择最满意的一套

## 代码结构说明

### 核心类说明

| 类名 | 文件 | 说明 |
|----|----|----|
| COCGenerater | main.py | 主界面，技能选择，极速生成程序，流程控制 |
| Choice2 | choice2.py | 分布生成程序 |
| Choice | choice3.py | 刺激生成程序 |
| Finale | finale.py | 五次八维生成选择，最终选择打印 |

### 数据流
resource.py 存储共享数据：

- skill_dict: 八大属性字典

- comment_sentence: 评价语句

- lyric: 彩蛋歌词

属性生成逻辑：

- 所有属性通过 (随机3D6)×5 计算

- 属性范围：15-90（实际15-90，3-18×5）
